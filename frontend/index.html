ABOUT ME

Hi, I'm Olli Tienvieri, and I am on my way to becoming your future cloud expert. I'm diving into Azure platform, and the hole is deep! As it's become evident that the learning journey is never-ending, I might as well enjoy the daily little victories, learn from the mistakes and keep on going even when things feel hard.

I believe in learning by doing, so as one step in my cloud learning journey, I built this Azure hosted resume website. The site uses Azure Blob storage's static website hosting, CosmosDB and Azure Functions to store and retrieve visitor counter data, Azure DNS and CDN to use custom domain and https, and GitHub Actions to build a CI/CD pipeline. The idea for the site came from A Cloud Guru Azure Resume Challenge, and you can read my documentation on this project on my GitHub
